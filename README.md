# Bot WhatsApp 100% GRATUITO

Usa Python + Flask + Google Apps Script.

## Comandos
- add João 2 bombom 1 bolo
- pag João 1 bombom
- ver João
- relatorio
- del João
- ajuda

## Valores
- Bombom = R$ 5
- Bolo = R$ 12
